# CDcall
本项目为本科大二安卓课程的期末项目，由本人独立完成。

本项目已上架酷安应用商店
商店页面：[通话记录查询-酷安](https://www.coolapk.com/apk/com.inntechy.a11039.cdcall）

### 功能介绍
用于统计与某联系人的通话总时长，以及各种情况的次数（如未接，拒接等），并生成用于分享的图片，可在图片二维码中加入自己输入的文本。
